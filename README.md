# billing-line

春日部アカデミー 月次引き落とし金額のLINE自動配信システム

## 概要

毎月の引き落とし金額を、春日部アカデミー公式LINEアカウント（@pwg1825g）から保護者一人一人に個別配信するシステム。Wordの差し込み印刷＋メール送信からの移行を目的とする。

## 主要機能

- **月次配信**：引き落とし日の約1週間前に、各家族へLINEで金額通知
- **配信タイプ対応**：自動引落（自）と直接持参（直）の2種類のテンプレートを使い分け
- **兄弟まとめ配信**：同一家族の兄弟分は1通にまとめて送信
- **Excel取込＋画面修正**：従来のExcelデータをそのまま取り込み、必要に応じて管理画面で個別修正
- **滞納フォロー**：引き落とし失敗時、理由（残高不足／ゆうちょ未処理）に応じた文面を自動生成

## システム構成

- **GAS**（Google Apps Script）：配信エンジン・データ管理
- **Googleスプレッドシート**：マスタデータ・配信ログ
- **GitHub Pages**：管理画面（ふくちさん用）・保護者LINE登録画面
- **LINE Messaging API**：配信
- **LINE Login API**：保護者LINE_USER_ID収集

## ブランチ運用
